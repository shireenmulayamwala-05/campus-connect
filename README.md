# CampusConnect

CampusConnect is a full-stack college community and campus management platform built using the MERN stack.  
It helps students connect, communicate, and manage campus-related activities through a modern web application.

---

## Features

- User Authentication
- Student Dashboard
- Admin Panel
- Email Integration
- Cloudinary Image Upload
- Responsive UI
- Fast Frontend with Vite
- REST API Backend

---

## Tech Stack

### Frontend
- React.js
- Vite
- CSS / Tailwind

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other Tools
- Cloudinary
- JWT Authentication
- Nodemailer / SMTP
